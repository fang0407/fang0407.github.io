HTML基本骨架

HTML 基本骨架是网页模板。

  • html：整个网页。
  • head：网页头部，存放给浏览器看的代码，例如 CSS。
  • body：网页主体，存放给用户看的代码，例如图片、文字。
  • title：网页标题。

HTML图像标签

作用：在网页中插入图片。

标签名：img （单标签，Image）。

标签示例：<img src=图片的URL>。

标签属性：

属性 作用 说明
src 图片路径 必选
alt 替换文本 图片无法显示的时候显示的文字
title 提示文本 鼠标悬停在图片上面的时候显示的文字
width 图片的宽度 值为数字，没有单位
height 图片的高度 值为数字，没有单位
  • 属性名=”属性值”
  • 属性写在尖括号里面，标签名后面，标签名和属性之间用空格隔开，不区分先后顺序。

HTML链接标签

作用：点击跳转到其他页面。

标签名：a（双标签，Anchor）。

标签示例：<a href="http://www.baidu.com">跳转到百度</a>。

标签属性：

属性 作用 说明
href 指定链接目标的URL 必选
target 指定链接如何在浏览器中打开 可选，常见的值包括 _blank（在新标签或窗口中打开链接）和 _self（在当前标签或窗口中打开链接）。
title 提示文本 鼠标悬停在图片上面的时候显示的文字
