Posted on 2023-12-07 Edited on 2023-12-30 In Janus

Janus 是⼀个开源的，通过 C 语⾔实现了对 WebRTC ⽀持的 Gateway；Janus ⾃身实现得很简单，提供插件机制来⽀持不同的业务逻辑，配合官⽅⾃带插件就可以⽤来实现⾼效的 Media Server 服务。
