Go HTTP的Swagger实现

Posted on 2023-12-24 Edited on 2023-12-31 In Golang

本文展示如何使用 http-swagger 包集成 Go net/http 包和 swagger。

使用步骤

  • 安装 swag 包。
  • 生成 docs 目录，在工作目录执行
  • 导入包。
  • 在 main 函数之前添加 swag 注释说明，示例如下：

title：应用程序的标题。

version：应用程序AP版本号。

description：应用程序的说明描述。

contact.name：公开的 API 的联系信息。

contact.url：指向联系信息的 URL。必须采用 URL 格式。

contact.email：联系人/组织的电子邮件地址。必须采用电子邮件地址的格式。

securitydefinitions.apikey：API 密钥授权。

具体注解字段说明：https://github.com/swaggo/swag#general-api-info

  • 编写每个函数的文档并将其放在函数本身之前。对于参数和响应，我们只需在文档中写下结构体名称，因为我们已经定义了结构体。
  • 最后我们必须在 main 函数中添加路由和 swagger 函数，这样我们就可以在浏览器中访问它了。
  • 设置完所有组件后，我们必须再次使用 swag 命令生成文档。
  • 最后，我们可以构建并运行该应用程序。我们可以访问 http://<host>:<port>/swagger/index.html 中的文档。
